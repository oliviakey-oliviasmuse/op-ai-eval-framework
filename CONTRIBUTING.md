# Contributing

Improvements welcome. Particularly:
- Domain-specific scenario templates (manufacturing, logistics, healthcare, energy)
- Additional scoring dimensions for specific industries
- Integration examples for existing agent frameworks (LangChain, AutoGen, etc.)
- Case studies from operational deployments

## Pull Request Process
1. Fork the repo
2. Add your contribution with clear documentation
3. Include rationale for any scoring changes
4. Link related issues

## Issues
Open an issue for: framework gaps, scenario templates for your industry, integration challenges.
